# alreadystillagain website

This is a bespoke portfolio project featuring a physics-based "Time" section, an immersive "Space" gallery, and a relational "Thought" flowchart.

## How to run the website

1. **Open Terminal** in VS Code (Go to Terminal -> New Terminal).
2. **Start the development server** by typing:
   ```bash
   npm run dev
   ```
3. **Open the link** provided in the terminal (usually `http://localhost:5173`) by holding `Command` and clicking it.

## Project Structure

- `/src/data/projects.js`: Contains the info for your work (titles, Vimeo IDs, thumbnails).
- `/src/components/TimeSection.jsx`: The "Hourglass" physics engine logic.
- `/src/App.jsx`: The main controller that handles navigation and the video modal.

## Commands to remember
- `npm run dev`: Starts the site.
- `Ctrl + C`: Stops the site in the terminal.
